# Student Task Board (Spec-Driven Development)

This project was built as part of **EPAM Lab 06** using the **SpecKit Framework**.

## Tech Stack
- **Framework:** React 19 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **State:** LocalStorage via Custom Hooks

## Methodology
This project follows **Spec-Driven Development (SDD)**. 
The implementation is driven by the specifications found in the `.specify/` directory.
